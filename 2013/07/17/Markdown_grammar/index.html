Write Markdown text

All my blog articles are written by Markdown. Here I want to introduce some grammers and tools on how to write the blog by Markdown. The pattern is really simple and beautiful.

1. Related installation

In order to build the blog, I installed

  • Python2.7.3 in Git Bash;

  • Pelican in Git Bash(which seems not to support Python2.6.5 on Ubuntu)

  • Pandoc on windows;

    http://johnmacfarlane.net/pandoc/index.html

  • MardownPad 2 on windows.

It is happy that all the things can be done on Windows, but turn on virtual machine for Ubuntu.

2. Basic grammars of Markdown

Firstly, I clone some samples of other’s articles which are written by Markdown from github. When I encounter new problems, I always refer the following links:

  • http://wowubuntu.com/markdown/#p

  • http://iout.in/archives/454.html

  • http://daringfireball.net/projects/markdown/syntax#autoescape

  • http://ued.taobao.com/blog/2012/07/getting-started-with-markdown/

  • http://wlog.cn/soft/why-use-markdown.html

3. Make use of Pandoc

Pandoc can tranfer .md into .pdf or *.html for previewing. There are some explanations as follow:

  • http://www.yangzhiping.com/tech/pandoc.html

In fact, I use one command under cmd by now.
pandoc .md -o .html

4 Write Markdown online

I don’t really like MardownPad on windows, as it’s too resource-consuming for me laptop. I just paste the codes on the following websites to preview the result.

  • http://benweet.github.io/stackedit/#

  • http://dillinger.io/

5 Update the blog

  • pelican -s settings.py
  • ssh -T git@github.com
  • git init
  • git add .
  • git commit -a -m “commit”
  • git remote add origin https://….
  • git push origin master

  • http://www.cnblogs.com/findingsea/archive/2012/08/27/2654549.html
